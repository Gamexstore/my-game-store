# Game Store - GitHub Deployment

This project has been converted to a **Static Website** so it can be hosted for free on GitHub Pages.

## How to Upload to GitHub

1.  **Create a Repository**: Go to [GitHub.com](https://github.com) and create a new repository (e.g., `my-game-store`).
2.  **Upload Files**:
    *   Upload all the files in this folder (`index.html`, `style.css`, `script.js`, `images/`, etc.) to your repository.
    *   *Note*: You do not need `server.js` or `package.json` for GitHub Pages.
3.  **Enable GitHub Pages**:
    *   Go to your repository **Settings** > **Pages**.
    *   Under **Source**, select `main` (or `master`) branch.
    *   Click **Save**.
4.  **Visit Site**: GitHub will give you a link (e.g., `https://yourname.github.io/my-game-store/`).

## Project Files
- `index.html`: The home page.
- `game.html`: The game details page.
- `download.html`: The human verification page.
- `style.css`: The site styling.
- `script.js`: The site logic.
- `images/`: The game images.
